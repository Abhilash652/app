Fix Engine

Recommended interventions

Balanced data repair

Increase representation for under-sampled groups and remove duplicate patterns that over-amplify dominant groups.

Proxy feature review

Flag features such as zip code or income bands that may indirectly encode protected attributes.

Threshold tuning by fairness target

Simulate threshold changes to reduce false negative gaps while keeping business accuracy within acceptable limits.
